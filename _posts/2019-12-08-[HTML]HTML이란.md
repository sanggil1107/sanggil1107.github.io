# HTML이란
---
HyperText Markup Language의 약자로 태그들을 이용하여 텍스트 이상의 요소를 정의하는 약속된 언어라고 할 수 있다.

## 기본구조
---

```html
<!DOCTYPE html>
<html>
    <head>
    </head>
    <body>
    </body>
</html>
```

## 주석
---
`<!--  -->` 와 같이 사용한다.
```html
<!-- 주석 내용 -->
```
