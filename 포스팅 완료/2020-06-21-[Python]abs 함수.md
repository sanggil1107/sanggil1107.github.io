
## abs 함수

`abs 함수` 는 인자로 들어온 값의 절대값을 구하는 함수이다.

```text
abs(x)
```

- x : 실수 타입 자료형
  
매개변수로 넣은 숫자가 변하는 것이 아니라 매개변수의 절대값이 반환된다.

<br>

### abs 함수 예제
---

```python
num1 = -99
num2 = 99
print(num1)
print(num2)

num3 = -99.99
num4 = 99.99
print(num3)
print(num4)

num5 = 0
num6 = 0.0
print(num5)
print(num6)
```
```text
99
99

99.99
99.99

0
0.0
```
