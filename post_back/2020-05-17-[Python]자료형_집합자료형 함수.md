



## 함수
---
### add - 값 1개 추가하기
---
`add(x)`
- x의 값을 집합 자료형에 추가한다.
```python
s1 = set('Hello')
s1.add(4)
s1
{4, 'e', 'o', 'l', 'H'}
```
### update - 값 여러 개 추가하기
---
`update(x)`
- 반복 가능한 x(리스트, 튜플, 문자열)를 집합 자료형에 추가한다.
```python
s1 = set('HI')
s1.update([4,5,6,7])
s1
{4, 5, 6, 7, 'I', 'H'}
```

### remove - 특정 값 제거하기
---
`remove(x)`
- x 값을 집합 자료형에서 제거한다.
```python
s1 = set('HELLO')
s1.remove('E')
s1
{'H', 'O', 'L'}
```
